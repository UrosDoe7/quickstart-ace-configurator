# OnDemand configuration with Monaco

You will create on demand configuration with monaco

- git clone 
      
      cd;
      git clone https://github.com/JLLormeau/OnDemand-Configuration-with-Monaco

- install monaco

      cd;cd OnDemand-Configuration-with-Monaco;
      wget https://github.com/dynatrace-oss/dynatrace-monitoring-as-code/releases/download/v1.5.3/monaco-linux-amd64;
      mv monaco-linux-amd64 monaco;
      chmod +x monaco;
